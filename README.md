# AttentiveFlow

AttentiveFlow is an agentic marketing workflow prototype for ecommerce lifecycle campaigns.

It converts messy marketing briefs into structured, personalized, cross-channel workflows across SMS, email, push, and RCS. The system uses specialized agents for campaign parsing, audience segmentation, message generation, compliance checking, experiment design, evaluation, and automated repair.

The goal is to demonstrate how AI agents can help lifecycle marketing teams move from campaign idea to testable workflow faster while improving quality, personalization, and compliance.

## Core Features

- Brief-to-workflow generation
- Audience segmentation from customer attributes
- SMS, email, push, and RCS message generation
- Compliance and channel eligibility checks
- A/B test design
- Campaign quality scoring
- Automated critique and repair loop
- Workflow JSON export
- Synthetic ecommerce event simulator
